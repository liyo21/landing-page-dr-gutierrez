# Landing Page — Dr. José Alfredo Gutiérrez

Landing page profesional desarrollada con Vue 3, TypeScript y Tailwind CSS v4.

## Tecnologías

- [Vue 3](https://vuejs.org/) con `<script setup>` (Composition API)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)

## Desarrollo

```bash
npm install
npm run dev
```

## GitHub Copilot

Este proyecto incluye instrucciones personalizadas para GitHub Copilot en `.github/copilot-instructions.md`. Para utilizarlo:

1. Instala la extensión **GitHub Copilot** en VS Code.
2. Inicia sesión con tu cuenta de GitHub que tenga acceso a Copilot.
3. Copilot leerá automáticamente las instrucciones del proyecto para ofrecer sugerencias contextualizadas.

> Para obtener acceso a GitHub Copilot, visita [github.com/features/copilot](https://github.com/features/copilot) y activa una suscripción en tu cuenta de GitHub.
